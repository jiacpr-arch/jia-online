# รูปประกอบควิซเกริ่นนำ (Teaser Quiz)

ไฟล์รูปจริงเป็น **WebP** (ย่อกว้าง 1200px คุณภาพ 82 — เล็ก ~75–120 KB/รูป) แสดงในควิซหน้าแรก สัดส่วนแสดงผล 16:10 (object-fit cover)
ถ้าไฟล์หาย/โหลดไม่ได้ ระบบจะแสดง **กล่อง emoji ประกอบอัตโนมัติ** แทน (ไม่มีรูปแตก)

| ไฟล์ | ใช้กับคำถาม | รูปที่ควรใช้ |
| --- | --- | --- |
| `q1.webp` | เจอคนหมดสติ ทำอะไรก่อน | คนกำลังประเมินความปลอดภัยรอบที่เกิดเหตุ |
| `q2.webp` | เบอร์ฉุกเฉิน 1669 | โทรศัพท์/รถพยาบาล/สายด่วน 1669 |
| `q3.webp` | กดหน้าอกตรงไหน | ตำแหน่งวางมือกลางหน้าอก |
| `q4.webp` | กดเร็วเท่าไร | มือกดหน้าอก / เมโทรนอม / จังหวะ |
| `q5.webp` | AED ใช้ได้ไหม | เครื่อง AED (เครื่องกระตุกหัวใจ) |

> อยากเปลี่ยนรูป: เซฟทับชื่อเดิม (WebP) หรือเปลี่ยนค่า `img` ใน `TEASER_QUIZ` (`src/App.jsx`)
> สคริปต์ย่อ: `Image.open(src).convert('RGB').resize((1200,h)).save(dst,'WEBP',quality=82,method=6)` (Pillow)
