# รูปประกอบควิซเกริ่นนำ (Teaser Quiz)

วางไฟล์รูปจริงทับชื่อด้านล่างนี้เพื่อให้แสดงในควิซหน้าแรก (สัดส่วนแนะนำ 16:10, JPG/PNG, กว้าง ~800px)
ระหว่างที่ยังไม่มีไฟล์ ระบบจะแสดง **กล่อง emoji ประกอบอัตโนมัติ** แทน (ไม่มีรูปแตก)

| ไฟล์ | ใช้กับคำถาม | รูปที่ควรใช้ |
| --- | --- | --- |
| `q1.png` | เจอคนหมดสติ ทำอะไรก่อน | คนกำลังประเมินความปลอดภัยรอบที่เกิดเหตุ |
| `q2.png` | เบอร์ฉุกเฉิน 1669 | โทรศัพท์/รถพยาบาล/สายด่วน 1669 |
| `q3.png` | กดหน้าอกตรงไหน | ตำแหน่งวางมือกลางหน้าอก |
| `q4.png` | กดเร็วเท่าไร | มือกดหน้าอก / เมโทรนอม / จังหวะ |
| `q5.png` | AED ใช้ได้ไหม | เครื่อง AED (เครื่องกระตุกหัวใจ) |

> เปลี่ยนชื่อ/ที่อยู่ไฟล์ได้ที่ค่า `img` ในตัวแปร `TEASER_QUIZ` ใน `src/App.jsx`
